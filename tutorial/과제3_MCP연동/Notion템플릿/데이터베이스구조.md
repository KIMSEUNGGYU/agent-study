# Notion 데이터베이스 구조 가이드

## 고객 관리 데이터베이스

### 속성 (Properties)

| 속성명 | 타입 | 설명 |
|--------|------|------|
| 고객명 | Title | 고객 이름 |
| 고객ID | Text | 고유 식별자 (C001, C002...) |
| 등급 | Select | VIP / Gold / Silver |
| 가입일 | Date | 고객 가입 날짜 |
| 최근구매일 | Date | 마지막 구매 날짜 |
| 총구매액 | Number | 누적 구매 금액 (원) |
| 담당자 | Person | 담당 영업사원 |
| 메모 | Text | 추가 메모 |

### Select 옵션

**등급:**
- VIP (색상: 빨강) - 총구매액 500만원 이상
- Gold (색상: 노랑) - 총구매액 200만원 이상
- Silver (색상: 파랑) - 그 외

---

## 매출 분석 데이터베이스

### 속성 (Properties)

| 속성명 | 타입 | 설명 |
|--------|------|------|
| 제품명 | Title | 판매된 제품 이름 |
| 날짜 | Date | 판매 날짜 |
| 카테고리 | Select | 제품 카테고리 |
| 수량 | Number | 판매 수량 |
| 단가 | Number | 개당 가격 |
| 매출액 | Formula | 수량 × 단가 |
| 담당자 | Person | 판매 담당자 |

### Select 옵션

**카테고리:**
- 전자기기 (색상: 파랑)
- 의류 (색상: 보라)
- 생활용품 (색상: 초록)

---

## 데이터 연동 가이드

### Google Sheets → Notion 연동 시

1. **CSV 파일을 Google Sheets에 업로드**
2. **Claude에게 요청**: "Google Sheets의 고객 데이터를 Notion 고객관리 데이터베이스에 추가해줘"
3. **데이터 검증**: 등급별 필터링으로 데이터 확인

### 활용 예시

```
# 분석 요청
"Notion의 매출 분석 데이터베이스에서 이번 달 전자기기 카테고리 매출 합계를 알려줘"

# 보고서 생성
"Notion의 고객 데이터를 분석해서 등급별 고객 현황 보고서를 만들어줘"

# 데이터 추가
"이 CSV 데이터를 Notion 데이터베이스에 새 행으로 추가해줘"
```
