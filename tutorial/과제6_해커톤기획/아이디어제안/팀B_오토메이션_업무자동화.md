# 오토메이션 - 반복 업무 제로 프로젝트

## 팀 정보
- **팀명**: 오토메이션
- **팀장**: 이서연 (기획팀)
- **팀원**: 한소희(마케팅), 윤지현(HR), 권도현(개발3팀)

## 아이디어 개요

### 문제 정의
각 부서에서 매주/매월 반복되는 정형화된 업무가 존재합니다.

**현황 조사 결과:**
- HR팀: 주간 출퇴근 현황 정리 (주 2시간)
- 마케팅팀: 주간 SNS 성과 리포트 (주 3시간)
- 기획팀: 주간 회의록 정리 및 배포 (주 1.5시간)
- 재무팀: 월간 경비 정산 취합 (월 8시간)

### 솔루션
n8n + Claude API를 활용한 업무 자동화 워크플로우 구축

**핵심 기능:**
1. 정형 데이터 자동 수집 (Google Sheets, Slack)
2. AI 기반 데이터 정리 및 요약
3. 리포트 자동 생성
4. Slack/이메일 자동 발송

### 기대 효과
- 부서당 주 평균 3시간 절감
- 휴먼 에러 방지
- 담당자 부재 시에도 리포트 발행

## 자동화 대상 업무

| 업무 | 현재 소요 시간 | 자동화 후 | 절감률 |
|------|--------------|----------|--------|
| 출퇴근 현황 정리 | 2시간/주 | 10분/주 | 92% |
| SNS 성과 리포트 | 3시간/주 | 15분/주 | 92% |
| 회의록 정리 | 1.5시간/주 | 5분/주 | 94% |
| 경비 정산 취합 | 8시간/월 | 30분/월 | 94% |

## 기술 스택
- **워크플로우**: n8n (Self-hosted)
- **LLM**: Claude API
- **데이터 소스**: Google Workspace, Slack API
- **알림**: Slack Webhook

## 개발 계획

| 단계 | 내용 | 담당 |
|------|------|------|
| 1 | 대상 업무 상세 분석 | 이서연 |
| 2 | n8n 워크플로우 설계 | 권도현 |
| 3 | HR팀 자동화 구현 | 윤지현 |
| 4 | 마케팅팀 자동화 구현 | 한소희 |
| 5 | 통합 테스트 및 문서화 | 전원 |

## 리스크 및 대응

| 리스크 | 영향도 | 대응 방안 |
|--------|--------|----------|
| API 권한 이슈 | 중 | 사전 IT팀 협의 |
| 데이터 형식 불일치 | 중 | 표준 템플릿 제공 |
| 자동화 오류 | 상 | 알림 + 수동 백업 |

## 데모 시나리오
1. Slack 메시지 → 자동 회의록 생성
2. Google Sheets 데이터 → 주간 리포트 생성
3. 생성된 리포트 Slack 자동 발송
