# 데이터크루 - 실시간 경영 대시보드

## 팀 정보
- **팀명**: 데이터크루
- **팀장**: 박준혁 (개발2팀)
- **팀원**: 정태윤(데이터팀), 오진우(데이터팀), 강예린(디자인팀)

## 아이디어 개요

### 문제 정의
현재 경영진 보고를 위해 여러 부서에서 수동으로 데이터를 취합하고 있습니다.

**현황:**
- 매출 데이터: ERP에서 수동 추출 (재무팀)
- 고객 데이터: CRM에서 별도 추출 (영업팀)
- 마케팅 성과: 각 채널별 수동 집계 (마케팅팀)
- 보고서 작성: 주 1회, 담당자 3명이 반나절 소요

### 솔루션
실시간 데이터 연동 경영 대시보드 구축

**핵심 기능:**
1. 다중 데이터 소스 자동 연동
2. 실시간 KPI 모니터링
3. AI 기반 이상 징후 감지
4. 자연어 질의 응답 (예: "이번 달 매출 top 5 제품은?")

### 기대 효과
- 보고서 작성 시간 90% 절감
- 실시간 의사결정 지원
- 데이터 기반 인사이트 자동 도출

## 대시보드 구성

| 섹션 | 포함 지표 | 데이터 소스 |
|------|----------|------------|
| 매출 현황 | 일/주/월 매출, 전년 대비 | ERP |
| 고객 현황 | 신규/이탈, NPS, 문의량 | CRM, CS시스템 |
| 마케팅 성과 | CAC, ROAS, 채널별 전환 | GA, 광고 플랫폼 |
| 운영 현황 | 재고, 배송, 반품률 | WMS |

## 기술 스택
- **프론트엔드**: Next.js + Recharts
- **백엔드**: FastAPI
- **데이터 파이프라인**: Apache Airflow
- **LLM**: Claude API (자연어 질의)
- **DB**: PostgreSQL + Redis (캐싱)

## 개발 계획

| 단계 | 내용 | 담당 |
|------|------|------|
| 1 | 데이터 소스 분석 및 스키마 설계 | 정태윤 |
| 2 | 데이터 파이프라인 구축 | 오진우 |
| 3 | 대시보드 UI/UX 디자인 | 강예린 |
| 4 | 프론트엔드 개발 | 박준혁 |
| 5 | 자연어 질의 기능 개발 | 오진우 |

## 데모 시나리오
1. 대시보드 메인 화면 시연
2. 드릴다운 (매출 → 제품별 → 일별)
3. 자연어 질의: "지난주 대비 매출 변화 알려줘"
