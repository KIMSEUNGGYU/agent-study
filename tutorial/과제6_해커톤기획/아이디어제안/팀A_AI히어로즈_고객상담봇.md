# AI 히어로즈 - 스마트 고객 상담봇

## 팀 정보
- **팀명**: AI 히어로즈
- **팀장**: 김민수 (개발1팀)
- **팀원**: 최유진(디자인), 배수아(CS팀), 임현우(개발3팀)

## 아이디어 개요

### 문제 정의
현재 CS팀은 하루 평균 200건 이상의 반복적인 문의를 처리하고 있습니다.
- 배송 조회: 35%
- 교환/환불 절차: 25%
- 제품 사용법: 20%
- 기타: 20%

이 중 70% 이상은 정형화된 답변으로 해결 가능한 문의입니다.

### 솔루션
Claude API를 활용한 지능형 고객 상담 챗봇을 개발합니다.

**핵심 기능:**
1. FAQ 기반 자동 응답
2. 주문 정보 연동 배송 조회
3. 복잡한 문의는 상담사 연결
4. 상담 내역 자동 요약 및 분류

### 기대 효과
- CS팀 업무 부담 50% 감소
- 고객 응답 시간 평균 30분 → 즉시 응답
- 24시간 상담 가능

## 기술 스택
- **LLM**: Claude API (Sonnet)
- **프론트엔드**: React + TypeScript
- **백엔드**: FastAPI
- **DB**: PostgreSQL

## 개발 계획

| 단계 | 내용 | 담당 |
|------|------|------|
| 1 | FAQ 데이터 정리 및 프롬프트 설계 | 배수아 |
| 2 | 챗봇 UI/UX 디자인 | 최유진 |
| 3 | 백엔드 API 개발 | 김민수 |
| 4 | 프론트엔드 개발 | 임현우 |
| 5 | 통합 테스트 | 전원 |

## 리스크 및 대응

| 리스크 | 영향도 | 대응 방안 |
|--------|--------|----------|
| API 응답 지연 | 중 | 캐싱 적용, 비동기 처리 |
| 환각(hallucination) | 상 | FAQ 기반 제한, 검증 로직 |
| 개인정보 노출 | 상 | 마스킹 처리, 로깅 최소화 |

## 데모 시나리오
해커톤 당일 시연할 3가지 시나리오:
1. 배송 조회 문의 처리
2. 교환 절차 안내
3. 복잡한 문의 → 상담사 연결
