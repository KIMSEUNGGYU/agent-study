# 서치마스터 - AI 사내 지식 검색

## 팀 정보
- **팀명**: 서치마스터
- **팀장**: 송민재 (개발1팀)
- **팀원**: 조성훈(개발2팀), 신유나(기획팀), 장미래(디자인팀)

## 아이디어 개요

### 문제 정의
사내 지식이 여러 시스템에 분산되어 있어 필요한 정보를 찾기 어렵습니다.

**현황:**
- Notion: 기획 문서, 회의록
- Confluence: 기술 문서
- Google Drive: 보고서, 자료
- Slack: 논의 내역, 공지

**문제점:**
- 같은 질문이 반복됨 ("이거 어디서 찾아요?")
- 신입사원 온보딩 시 정보 찾기 어려움
- 퇴사자의 암묵지 유실

### 솔루션
RAG 기반 통합 사내 검색 시스템

**핵심 기능:**
1. 다중 소스 문서 인덱싱
2. 자연어 검색 + 출처 표시
3. 관련 문서 추천
4. 신규 문서 자동 인덱싱

### 기대 효과
- 정보 검색 시간 70% 단축
- 반복 질문 감소
- 조직 지식 자산화

## 검색 대상 소스

| 소스 | 문서 유형 | 예상 문서 수 |
|------|----------|-------------|
| Notion | 기획서, 회의록 | ~2,000건 |
| Confluence | 기술 문서, API 스펙 | ~500건 |
| Google Drive | 보고서, 발표자료 | ~3,000건 |
| Slack | 주요 채널 메시지 | ~10,000건 |

## 기술 스택
- **임베딩**: OpenAI text-embedding-3-small
- **벡터 DB**: Pinecone
- **LLM**: Claude API (답변 생성)
- **프론트엔드**: React
- **백엔드**: FastAPI
