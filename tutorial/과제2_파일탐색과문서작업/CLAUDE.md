# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트 개요

사내 교육 플랫폼 구축 프로젝트의 문서 저장소입니다. 코드가 아닌 기획/관리 문서 중심 저장소입니다.

- **기간**: 2024년 1월 ~ 3월 (3개월)
- **총 예산**: 5,000만원
- **기술 스택**: React + Node.js (2주차에 확정)
- **PM**: 김민수 (경영지원팀)

## 폴더 구조와 역할

```
프로젝트파악/
  기획/          # 프로젝트 개요, 요구사항 정의서, 일정 계획
  회의록/        # 회의록 (파일명: 회의록_MMDD_제목.md)
  보고서/        # 주간보고 (파일명: 주간보고_N주차.md)
  데이터/        # 예산/일정 현황 CSV

프로젝트설정/
  정리전/        # 미가공 메모·초안 (작업 전 임시 보관)
  완료/          # 정리 완료된 문서
```

## 요구사항 구조

요구사항은 `프로젝트파악/기획/요구사항_정의서.md`에 ID 체계로 관리합니다.

| 접두사 | 영역 |
|--------|------|
| UR-    | 사용자 관리 |
| CR-    | 강의 관리 |
| LR-    | 학습 관리 |

1단계(필수) / 2단계(선택) 우선순위로 구분됩니다. 모바일 지원은 2단계입니다.

## 주요 외부 연동

- SAP HR (인사 시스템)
- SSO (사번 로그인) — 연동 가능 여부 IT팀 협의 진행 중
- 사내 메일 (교육 알림)
- DRM (콘텐츠 보호) — 외부 솔루션 검토 중

## 문서 작성 규칙

- 회의록: `회의록_MMDD_제목.md` 형식으로 `회의록/` 폴더에 저장
- 주간보고: `주간보고_N주차.md` 형식, 작성자는 PM(김민수)
- 초안/메모는 `정리전/`에 먼저 보관 후 정리 완료 시 `완료/`로 이동
- 날짜 형식: `YYYY년 MM월 DD일` 또는 `MM/DD`
