# CS점수 분석 스킬

## 워크플로우

1. 프로젝트 내 `CS점수_*.csv` 파일을 Glob으로 탐색
2. `scripts/analyze_cs.py` 스크립트를 실행하여 분석 보고서 생성
3. 결과를 `분석/` 폴더에 마크다운으로 저장

## CSV 형식

분석 대상 CSV 파일은 다음 컬럼 구조를 따름:

```
주차,상담만족도,문제해결률,응대친절도,전체평균
1주차,4.0,82%,4.2,4.10
```

- 파일명 규칙: `CS점수_N월.csv` (월 정보를 파일명에서 추출)
- 파일 위치: `데이터/N월/` 디렉토리

## 스크립트 사용법

```bash
python3 <skill-path>/scripts/analyze_cs.py <출력경로> <CSV파일1> [CSV파일2] ...
```

예시:
```bash
python3 <skill-path>/scripts/analyze_cs.py 분석/CS점수_분석.md 데이터/1월/CS점수_1월.csv 데이터/2월/CS점수_2월.csv
```

## 분석 항목

- **월 평균 점수**: 각 지표(상담만족도, 문제해결률, 응대친절도, 전체평균)의 월별 평균
- **주차별 추이**: 주차별 각 지표 변화 테이블
- **최고/최저점**: 각 지표의 최고·최저 값
- **월간 비교**: 2개월 이상 데이터 시 월별 평균 비교 및 추이 요약

## 출력

결과는 `분석/CS점수_분석.md`에 마크다운 테이블 형태로 저장.
